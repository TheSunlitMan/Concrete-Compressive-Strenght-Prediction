
# Прогнозирование прочности бетона на сжатие

## Описание проекта
Прогнозирование прочности бетона на сжатие на основе данных о компонентном составе смеси с использованием методов машинного обучения.

### Актуальность
Повышение точности прогнозирования прочности бетона позволяет оптимизировать расход материалов и обеспечить надежность строительных конструкций, что напрямую влияет на экономическую эффективность и безопасность в строительной отрасли.

## Цель работы
Разработать и сравнить точность моделей машинного обучения для прогнозирования прочности бетона на сжатие на основе данных о компонентном составе смеси, интегрировать лучшую модель в приложение с графическим интерфейсом.

## Задачи исследования
1. Провести анализ и предобработку данных о компонентах бетонной смеси
2. Реализовать и обучить различные модели машинного обучения
3. Сравнить точность моделей и выбрать оптимальную
4. Разработать приложение с графическим интерфейсом
5. Протестировать работоспособность решения

## Выбор финальной модели
**Kernel Ridge Regression** была выбрана в качестве финальной модели по следующим причинам:
1. **Высокая точность**: R² = 0.929, что сопоставимо с лучшими ансамблевыми методами
2. **Вычислительная эффективность**: время обучения 0.21s против 3.63s у XGBoost
3. **Стабильность**: наименьший разброс результатов при кросс-валидации
4. **Оптимальная сложность**: соответствует объему данных (1030 наблюдений)

## Установка и запуск

### Предварительные требования
- Python 3.8+
- pip (менеджер пакетов Python)

# Установка зависимостей
